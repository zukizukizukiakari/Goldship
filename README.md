她们如果真的是拍小电影的话会被人骂的吧？！训练员拆着套套，有点跑偏的想。  
  
## 06
吹完了不太对劲的气球之后，黄金船又从床头柜里翻出了眼罩手铐之类的一系列道具，不过漂亮的赛马娘看起来对这些兴致缺缺，最后目光还是放回了胡萝卜小玩具上。  
“这个要怎么用？”  
训练员已经对黄金船的0经验见怪不怪了：“你倒是看看说明书啊，不过这种东西一般都是用开关控制吧？”  
这个胡萝卜造型还挺可爱的，训练员拿着它左看看右看看，结果黄金船找到一个遥控按了按钮，小胡萝卜就在训练员手里猛的震动起来，把她给吓了一跳：“哇！等等黄金船你先把它给关了！”  
黄金船听见小胡萝卜震动发出的嗡鸣声，似乎更加感兴趣了些：“你现在用起来就好了啊。”  
“拜托，要用也该是给你用。”训练员拿着震动的小胡萝卜不知所措，该灭火的正主压着她一个可怜的菜鸟使劲儿欺负，到底是什么人间疾苦，“快关掉啦，我的手都被这玩意震麻了。”  
黄金船不太情愿地又按下了按钮，结果震动换了个模式，她一脸新奇地多按了几下，看着小胡萝卜的震动换了几种频率和强度，最后才停了下来，她仿佛打开了新世界的大门：“这个很有意思嘛！”

“这个，要放进里面吗？”零经验黄金船向训练员取经中。  
训练员努力地回想着小电影里的情节，有些怀疑地摇了摇头：“不一定？演员们都是身经百战的选手，我们这种出道赛群体强行放了反而会痛吧？”  
黄金船的保健体育还是有认真学的，闻言似乎觉得有些可惜：“不能一脚踹开闸门啊……”

训练员总觉得不会是什么好事，但还是半信半疑的靠了过去，黄金船立马就露出了得逞的笑容，一把搂住训练员的腰不让她逃跑：“好东西要分享！不如我们一起玩啦。”  
黄金船不让训练员乱动，把小胡萝卜夹在了她们之间，小胡萝卜本就是经过特殊设计的，这位置也是正好，训练员都还没来得及阻止，黄金船就已经按下了遥控的按钮——

训练员攀着黄金船的肩膀，忍着不要发出奇怪的声音来：“你，你把震动调弱点？”  
黄金船这会儿迷之老实，还真乖乖的把震动调弱了，她们同时松了口气，发现彼此同步之后又莫名其妙的笑了起来。

“现在是炒面完售加上出海满载而归的快乐！”  
都有感觉了能不能放弃怪比喻啊，训练员觉着有些好笑，结果猝不及防的被黄金船给亲了。  
有小胡萝卜的加持，现在的状况也算是不错的开局了，黄金船的本能可比训练员强得多，很快就熟练起来，该到终盘冲刺阶段的时候黄金船也不含糊，很精准的调高了小胡萝卜的强度——

黄金船笑了起来，故意凑到训练员耳边说起了悄悄话：“嘛……我觉得挺重要的哦。”

这次应该是同时冲线，大家都是第一名。  
“黄金船？”感觉贴在训练员耳边的呼吸比先前要重，训练员摸着黄金船的背小声问道，“还好吗？”

黄金船的呼吸又平稳下来，抬起脸和训练员抵着额头，眼睛亮得惊人，竖起来的马尾也垂了下来，不怀好意地绕到训练员的腰侧摇来摇去：“再来一次？”

训练员的表情和语调都变了：“唯独这个不想要啊！”

有黄金船在的场合肯定是不会安静的，训练员走着神和她有一句没一句的聊着天，最后她们再次来到了这个浴缸。和之前不太一样的是，这会儿训练员已经放弃了节操和形象，瘫在黄金船的怀里随她抱了。  
训练员放飞归放飞，不过关键的问题还是要问的：“感觉怎么样？”

“你这就是单纯的想去玩吧？！”  
“放心吧，会带你去的！”黄金船高兴地给训练员的头发打了两个蝴蝶结，“现在你就是本黄金船大人的家属了，家属要随行哦~”  
喜提家属身份的训练员翻了个白眼：“我怎么就荣升家属了？野营好累我不去。”  
“啥，我们都这样那样了，训练员你居然想不认账？”黄金船语气震惊，接着就开始假哭，“嘤嘤嘤训练员你好渣！居然睡过人家就翻脸不认人了，明明在床上的时候那么热情！”  
训练员鸡皮疙瘩掉了一地，她确实很想假装无事发生过来着：“别说得我像个负心汉啊！说到底与其说是我睡了你，还不如说是你睡了我……”  
“喂，我可是在好好负责呢。”黄金船不满地戳了戳训练员的脸，语气平平的，“你给我配合一点啊。”  
训练员干啥啥不行认怂第一名：“对不起，你负责吧。”  
黄金船一点也不买账，很不客气地揉起了训练员的耳朵：“不行，你一点都不诚心！”  
什么赛马娘，这分明是她大爷，训练员暗自腹诽着，心里还是有些惆怅的。她一开始和黄金船搭档，是因为受到了来自前辈训练员的打压，而黄金船却选择了初出茅庐的她。她们的相处方式也和其他的搭档不一样，很多时候黄金船都有自己的主张，训练员制定的计划，黄金船看心情挑着做，有些比赛也不是训练员去决定是否参加，而是黄金船说想要跑才让她去报的。她们确实是搭档，但也总是在无形的较劲，输赢也没有定数，彼此并不能轻易地取得压倒性胜利。  
发生了这样的意外事件，这种互相扶持着拳打脚踢的关系就有点微妙起来了，训练员对黄金船有没有超出负责的赛马娘这一范围的感情，她自己其实不是很清楚，而且她也不觉得黄金船就是明白的——外星马娘会爱上地球人吗？  
“在这之前，学院禁止训练员和赛马娘交往过密的吧？”  
纠结了半天，训练员忽然想起这么一茬。  
“但是好像没有阻止过亲密行为啊？”黄金船自己体验过了，对于她认识的赛马娘出现反常的行为也就理解了，“来旅馆这种事貌似挺常见的——这间旅馆的事是大家一起吃午饭的时候告诉我的。”  
训练员大受震撼，甚至有点想八卦：“你和谁吃午饭了？”  
黄金船报了几个名字，训练员想了下，表情又愁苦起来，训练员都是男性啊，只能说她俩太不对劲了。  
不过这想破了脑袋好像也没什么结果，而且黄金船也意外的放过了训练员：“算啦，100年之后记得和我一起去宇宙旅行就好！”  
“100年……咱们的骨灰盒一起上太空吗。”训练员还是不能放弃做吐槽役，“还是灵魂免费遨游太空？”  
黄金船对这个话题相当感兴趣：“路过太阳的时候灵魂也许会被热到融化……”  
话题被扯远，训练员回头看了眼兴致勃勃的黄金船，最后还是笑了起来。  
还真是捉摸不定的思春期啊。
